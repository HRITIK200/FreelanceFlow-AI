# FreelanceFlow AI

FreelanceFlow AI is a full-stack freelance management platform that helps freelancers manage clients, projects, invoices, payments, and activity logs from a single dashboard.

## Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Protected Routes

### Client Management
- Add Clients
- Edit Clients
- Delete Clients
- Search Clients

### Project Management
- Create Projects
- Update Projects
- Delete Projects
- Track Project Status

### Invoice Management
- Create Invoices
- Mark Paid/Pending
- Download Invoice PDF
- Send Invoice Email

### Dashboard
- Total Clients
- Total Projects
- Revenue Overview
- Activity Tracking
- Business Analytics

### Activity Logs
- Client Activities
- Project Activities
- Invoice Activities
- Timeline Tracking

## Tech Stack

### Frontend
- React.js

### Backend
- Node.js
